基于 Qt 的多媒體播放器(Qmmp)。

此程式是一個音樂播放器，程式的編寫基于 Qt 庫。

主要功能：

  • 未封包的 winamp 皮膚支援；
  • 插件支援；
  • MPEG1 1/2/3 層支援；
  • Ogg Vorbis 支援；
  • native FLAC 支援；
  • Musepack 支援；
  • WMA 支援；
  • ALSA 音響匯出；
  • JACK 音響匯出。

要求：

  • OS GNU Linux;
  • Qt 版本 >= 4.2;
  • MAD;
  • Ogg Vorbis;
  • FLAC >= 1.1.3;
  • ALSA >= 1.0.1;
  • TagLib >= 1.4.
  • libmpcdec >= 1.2.6
  • Jack >= 0.102.5
  • FFmpeg >= 0.4.9-pre1
