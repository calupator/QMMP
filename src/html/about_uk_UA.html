Qt-based Multimedia Player (Qmmp)

Ця програма є аудіо-програвачем, написаним з використанням бібліотеки Qt. Програма має інтерфейс, аналогічний winamp чи xmms.

Основні можливості програми:

  • підтримка тем winamp в розпакованому вигляді;
  • підтримка модулів (плагінів);
  • підтримка файлів MPEG1 layer 1/2/3;
  • підтримка файлів Ogg Vorbis;
  • підтримка файлів Native FLAC;
  • підтримка файлів Musepack;
  • підтримка файлів WMA;
  • виведення звуку через ALSA;
  • виведення звуку через Jack.

Для роботи необхідно:

  • операційна система GNU Linux;
  • Qt версії >= 4.2;
  • MAD;
  • Ogg Vorbis;
  • FLAC версії >= 1.1.3;
  • ALSA версії >= 1.0.1;
  • TagLib версії >= 1.4.
  • libmpcdec версії >= 1.2.6
  • Jack версії >= 0.102.5
  • FFmpeg версії >= 0.4.9-pre1
