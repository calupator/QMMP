Qt-basierender Multimedia Player (Qmmp)

Dieses Programm ist ein Audio-Player, der die Qt-Bibliotheken verwendet.

Wichtigste Merkmale:

  • unterstützt unkomprimierte Winamp skins;
  • Modularer Aufbau dank Plugin-Unterstützung;
  • unterstützt MPEG1 layer 1/2/3;
  • unterstützt Ogg Vorbis;
  • unterstützt nativen FLAC;
  • unterstützt Musepack;
  • unterstützt WMA;
  • unterstützt ALSA Sound Ausgabe;
  • unterstützt JACK Sound Ausgabe.

Voraussetzungen:

  • OS GNU Linux;
  • Qt >= 4.2;
  • MAD;
  • Ogg Vorbis;
  • FLAC >= 1.1.3;
  • ALSA >= 1.0.1;
  • TagLib >= 1.4.
  • libmpcdec >= 1.2.6
  • Jack >= 0.102.5
  • FFmpeg >= 0.4.9-pre1
